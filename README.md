# Recetario 

Este recetario de cocina es un espacio donde puedes consultar y almacenar las recetas que prefieras, con variedad de opciones para todos los paladares.

----

## Tecnologías

- tailwind
- Vite + React
- react-hook-form
- react-router-dom
- sweetalert2
- js-cookie
- axios
- node.js
- express
- bcryptjs
- cloudinary
- mongoose
- jsonwebtoken
- zod
- fs-extra
- morgan
- mongoDB

----

## funcionalidades

- guardar, editar y eliminar la receta
- guardar una imagen 
- buscador
- creación de usuario
